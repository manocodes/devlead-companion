# DevLead Companion (AI-powered engineering management hub)#
A web app for engineering managers and leads to:
- Track team members, skills, goals, and 1:1 notes
- Plan and track initiatives/projects
- Store docs (designs, retros, feedback, etc.) and query them via RAG
- Get AI assistance for:
  - Drafting 1:1 agendas and feedback
  - Summarizing project docs and risks
  - Generating action items from meeting notes
 
Built it as a multi-tenant SaaS (simple multi-user) with:
- Front-end SPA
- Back-end API
- Relational DB
- Vector store (for RAG)
- Cloud deployment
- Basic observability and CI/CD
